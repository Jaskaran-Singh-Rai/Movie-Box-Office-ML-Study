# Movie Box Office Trend Study
A Machine Learning project analyzing the factors that drive financial success in the film industry.

## Features
* **Data Cleaning:** Processing JSON-formatted genre and cast data.
* **Visualizations:** * Histogram (Revenue Distribution)
  * Bar Graph (Genre Performance)
  * Pie Chart (Language Market Share)
  * Heatmap (Correlation Analysis)
* **ML Model:** Random Forest Regressor to predict movie revenue.

## How to Run
1. Open the `.ipynb` file in Google Colab.
2. Upload the provided CSV datasets.
3. Run all cells.
